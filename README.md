<h1 align="center">🎰 CasinoBlocker</h1>
<p align="center">
  <strong>Автоматическая блокировка сайтов казино и букмекеров на Windows</strong>
</p>

<p align="center">
  Утилита, которая отслеживает открытые вкладки браузера и блокирует доступ к азартным сайтам через файл hosts.
</p>

---

## ✨ Возможности

- **Автоматическое обнаружение** — сканирует заголовки окон и определяет сайты казино по ключевым словам
- **Блокировка через hosts** — добавляет домены в `C:\Windows\System32\drivers\etc\hosts`
- **Поддержка Chrome/Edge** — извлекает URL из адресной строки браузера
- **Автозапуск** — создаёт задачу в Планировщике Windows для запуска при входе в систему
- **Безопасный список** — поисковики (Google, Yandex, Bing и др.) не блокируются
- **Многоязычность** — распознаёт ключевые слова на русском и английском

---

## 🚀 Быстрый старт

### Требования

- **Windows** 10/11
- **Python** 3.10+
- Браузер **Chrome** или **Edge**

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/sk1tezz/CasinoBlocker.git
cd CasinoBlocker

# Установка зависимостей
pip install -r requirements.txt
```

### Запуск

> ⚠️ **Важно:** первый запуск — **от имени администратора** (для записи в hosts и настройки автозапуска).

```bash
python main.py
```

Скрипт будет работать в фоне, проверяя окна каждую секунду.

---

## 📦 Сборка в .exe

Для запуска без установленного Python:

```bash
pip install pyinstaller
build.bat
```

Готовый файл: `dist\CasinoBlocker.exe`

## ⚙️ Как это работает

```
┌─────────────────────────────────────────────────────────────┐
│  1. Сканирование окон (pygetwindow)                          │
│     → Поиск заголовков с ключевыми словами (casino, казино…) │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  2. Определение браузера по формату "Название - Chrome"     │
│     → Извлечение URL из адресной строки (pywinauto)         │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  3. Проверка домена                                          │
│     → Игнор поисковиков (google.com, yandex.ru…)             │
│     → Добавление в hosts, если ещё не заблокирован           │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  4. Сброс DNS-кеша (ipconfig /flushdns)                      │
└─────────────────────────────────────────────────────────────┘
```

## 🔧 Зависимости

| Пакет | Назначение |
|-------|------------|
| `pygetwindow` | Получение списка окон и их заголовков |
| `pywinauto` | Доступ к элементам UI браузера (адресная строка) |
| `psutil` | Вспомогательные системные операции |

## ⚠️ Ограничения

- Работает только с **Chrome** и **Edge** (формат заголовка `Страница - Chrome`)
- Требуются **права администратора** для записи в hosts
- Блокировка применяется ко всей системе (все браузеры и приложения)
